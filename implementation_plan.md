## 2. コアコンセプト：ゴール指向の逆算推論

```
問題 → ゴール特定 → パラメータ分解 → 導出計画 → 計算実行 → 回答
```

1. **ゴール特定**: 問題が求めている「答え」は何か？
2. **パラメータ分解**: その答えを得るには何が必要か？
3. **導出計画**: 各パラメータをどう計算するか？ ← RAGから知識取得
4. **計算実行**: 計画に従い計算

---

## 3. 分野別RAG

### 構造
公式 + 適用条件をセットで格納:
```
例: 余弦定理
  公式: a² = b² + c² - 2bc·cosA
  適用条件: 三角形で、2辺と挟角がわかっている
```
条件に当てはまらなければ使わない、と判別可能。

### 優先順位
1. **幾何**: まずここから構築（正弦/余弦定理、円周角等）
2. 他分野: テストしながら必要に応じて追加
